# Random Quote Generator

**Автор:** [дильбар]
**Дата:** 02.05.2026

## Описание проекта

Графическое приложение для генерации случайных цитат с возможностью фильтрации по автору и теме. История сгенерированных цитат сохраняется в файл JSON и загружается при запуске.

## Возможности

- Генерация случайной цитаты.
- Отображение истории сгенерированных цитат.
- Фильтрация по автору и теме.
- Сохранение и загрузка истории в JSON.
- Проверка корректности ввода (пустые строки).
- Использование Git для контроля версий.

## Инструкция по запуску

1. Установите Python 3.
2. Клонируйте репозиторий:
   ```
   git clone https://github.com/[ваш_логин]/Random-Quote-Generator.git
   ```
3. Перейдите в папку проекта:
   ```
   cd Random-Quote-Generator
   ```
4. Запустите приложение:
   ```
   python main.py
   ```
5. Используйте интерфейс для генерации и фильтрации цитат.
6. История сохраняется автоматически при нажатии кнопки «Сохранить историю».

# Random Quote Generator

**Автор:** дильбар

## Описание
Приложение для генерации случайных цитат с фильтрацией по автору и теме, сохранением истории в JSON и загрузкой обратно.

## Возможности
- Генерация случайной цитаты.
- Отображение истории.
- Фильтрация по автору и теме.
- Добавление новых цитат с проверкой.
- Сохранение и загрузка истории.

## Примеры использования
1. Запустите main.py.
2. Нажмите «Сгенерировать цитату».
3. Используйте фильтры для поиска.
4. Добавьте свою цитату через форму.
5. История сохраняется при выходе.
